# Set navigation to gesture-based

> **Path:** Settings › Display › Navigation bar › Swipe gestures

## Why

Reclaims screen real estate on the 6.9-inch display and feels more natural.

## Steps

1. Open Settings
2. Tap Display
3. Tap Navigation bar
4. Select Swipe gestures

> 💡 **Tip:** Swipe up from bottom to go home, swipe up and hold for recent apps, swipe from sides to go back.
