# Enable Developer Options

> **Path:** Settings › About phone › Software information › Build number ×7

## Why

Enables USB debugging, animation speed control, and background process limits.

## Steps

1. Open Settings
2. Tap About phone
3. Tap Software information
4. Tap Build number 7 times rapidly
5. Enter PIN when prompted

> 💡 **Tip:** Developer Options appears at the bottom of the main Settings list after activation.
