# Set up Circle to Search

> **Path:** Settings › Display › Navigation bar › Circle to Search

## Why

Instantly search products, translate text, or identify objects without switching apps.

## Steps

1. Open Settings
2. Tap Display
3. Tap Navigation bar
4. Toggle on Circle to Search

> 💡 **Tip:** Hold the Home button (gesture bar) to activate Circle to Search from any app.
