# Customize Circle to Search

> **Path:** Settings › Display › Navigation bar › Circle to Search

## Why

Provides an alternative navigation method for S25 Ultra owners

## Steps

1. Open Settings
2. Tap Display
3. Tap Navigation bar
4. Toggle Circle to Search on
5. Confirm the navigation bar has changed

> 💡 **Tip:** You can switch back to the standard navigation bar if needed
