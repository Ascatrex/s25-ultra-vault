# Set up Secure Folder

> **Path:** Settings › Security and privacy › Secure Folder

## Why

Keep banking apps, private photos, and work documents completely separate and protected.

## Steps

1. Open Settings
2. Tap Security and privacy
3. Tap Secure Folder
4. Sign in with Samsung account
5. Set a separate lock (PIN, pattern, or biometric)

> 💡 **Tip:** Install separate copies of apps in Secure Folder — they run independently with separate logins.
