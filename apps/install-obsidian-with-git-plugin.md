# Install Obsidian with Git plugin

> **Path:** Play Store › Search › Obsidian › Install

## Why

Your daily agent reports and product recommendations are in the vault — read them anywhere.

## Steps

1. Install Obsidian from Play Store
2. Install the Obsidian Git community plugin in desktop first
3. Open the vault in Obsidian mobile (point to local folder or remote GitHub)
4. Enable auto-pull in Obsidian Git settings (every 10 min)
