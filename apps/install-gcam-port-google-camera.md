# Install Gcam Port (Google Camera)

> **Path:** Settings › Apps › Special access › Install unknown apps

## Why

Night Sight dramatically outperforms Samsung's default night mode in low-light conditions.

## Steps

1. Download a Gcam port for SM-S938B from a trusted source (e.g. Celsoazevedo.com)
2. Open Settings › Apps › Special access › Install unknown apps
3. Allow install from your file manager
4. Install the Gcam APK and open it
5. Import the recommended config XML for S25 Ultra

> 💡 **Tip:** Keep Samsung Camera for video — Gcam video quality is not as good on Samsung hardware.
