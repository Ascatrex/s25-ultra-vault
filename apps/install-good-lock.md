# Install Good Lock

> **Path:** Galaxy Store › Search › Good Lock › Install

## Why

Good Lock modules unlock One UI customization options not available in standard Settings.

## Steps

1. Open Galaxy Store (pre-installed)
2. Search "Good Lock"
3. Install Good Lock main app
4. Inside Good Lock, install: LockStar (lock screen), QuickStar (status bar), MultiStar (multi-window)

> 💡 **Tip:** Good Lock is only available in certain regions — use a VPN if not showing in your region.
